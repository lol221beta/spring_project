Тематика задания 8.1

Выбранная тема: Оборудование парка развлечений (таблица equipment).

Базовая структура таблицы по заданию: id, item_name, purchase_date, status. В рамках данного проекта эта таблица расширена до справочника аттракционов парка с дополнительными полями (возрастные и ростовые ограничения, вместимость, время работы и т.д.), но логически соответствует теме «Оборудование» из варианта 7.

Опыт работы с технологиями

В процессе разработки данного проекта были изучены и применены современные технологии веб-разработки на платформе Java. Проект реализован с использованием Spring Boot для создания полнофункционального веб-приложения, Spring Security для обеспечения безопасности, Spring Data JPA и Hibernate для работы с базой данных. Интерфейс разработан с использованием Thymeleaf и современного CSS с применением принципов дизайна, вдохновленных iOS. В процессе работы были освоены принципы трехзвенной архитектуры, RESTful API, объектно-реляционного отображения (ORM) и безопасной разработки приложений.
